# SQL Insight — System Architecture

## Overview

SQL Insight is a Spring Boot library that provides real-time insights into SQL queries generated by Hibernate/JPA. It captures query execution information and exposes metrics through a developer dashboard.

The architecture is modular and separated into backend monitoring components and a web dashboard.

---

## High Level Architecture

Application
   ↓
Hibernate Query Execution
   ↓
SQL Insight Interceptor
   ↓
Query Collector
   ↓
Query Analyzer
   ↓
Metrics Engine
   ↓
REST API
   ↓
Dashboard UI

---

## Modules

### sql-insight-core

Core monitoring engine.

Responsibilities:
- SQL interception
- Query data models
- Query collector
- Query analyzer
- Metrics engine

No Spring dependencies.

---

### sql-insight-annotations

Contains custom annotations for method-level monitoring.

Includes:
- @QueryTrace
- @QueryLabel
- @SlowQueryAlert
- @DisableQueryTracking

---

### sql-insight-spring-boot-starter

Spring Boot integration module.

Responsibilities:
- Auto configuration
- Property loading
- Register Hibernate interceptors
- Enable dashboard APIs

---

### sql-insight-dashboard

REST API for dashboard data.

Provides endpoints for:

- metrics
- query logs
- slow queries
- N+1 detection

---

### sql-insight-ui

Frontend dashboard.

Responsibilities:

- Visualize query metrics
- Display recent queries
- Show slow query alerts
- Display N+1 warnings

---

## Core Components

### QueryInterceptor

Intercepts SQL queries using Hibernate StatementInspector.

Responsibilities:

- Capture SQL query
- Trigger query tracking
- Attach context information

---

### QueryCollector

Stores captured queries in memory.

Implementation:

ConcurrentLinkedQueue<QueryInfo>

Limits:

maxQueries = 1000

Old queries are automatically removed.

---

### QueryAnalyzer

Processes collected queries.

Responsibilities:

- slow query detection
- query frequency tracking
- N+1 detection

---

### MetricsEngine

Aggregates statistics.

Metrics:

- totalQueries
- avgExecutionTime
- slowQueries
- nPlusOneWarnings

---

## Data Flow

1. Hibernate executes SQL query
2. Interceptor captures query
3. QueryCollector stores query
4. QueryAnalyzer evaluates query
5. MetricsEngine updates metrics
6. Dashboard API exposes data
7. React UI renders dashboard
